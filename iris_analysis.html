Task 1: Load and Explore the Dataset

In[5]:
# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.datasets import load_iris

# Load Iris dataset using sklearn
try:
    iris = load_iris()
    df = pd.DataFrame(iris.data, columns=iris.feature_names)
    df['species'] = iris.target
    df['species'] = df['species'].map({i:name for i, name in enumerate(iris.target_names)})
except Exception as e:
    print("Error loading dataset:", e)

Task 3: Data Visualization

1. Line Chart showing average petal length per species

In[17]:
grouped['petal length (cm)'].plot(kind='line', marker='o', title='Average Petal Length per Species')
plt.ylabel('Petal Length (cm)')
plt.xlabel('Species')
plt.grid(True)
plt.show()

2. Bar Chart average sepal width per species

In[20]:
grouped['sepal width (cm)'].plot(kind='bar', color='skyblue', title='Average Sepal Width per Species')
plt.ylabel('Sepal Width (cm)')
plt.xlabel('Species')
plt.xticks(rotation=0)
plt.grid(True)
plt.show()

3. Histogram distribution of petal length

In[3]:
plt.hist(df['petal length (cm)'], bins=20, color='orange', edgecolor='black')
plt.title('Distribution of Petal Length')
plt.xlabel('Petal Length (cm)')
plt.ylabel('Frequency')
plt.grid(True)
plt.show()

4. Scatter Plot sepal length vs petal length

In[24]:
sns.scatterplot(data=df, x='sepal length (cm)', y='petal length (cm)', hue='species')
plt.title('Sepal Length vs Petal Length by Species')
plt.grid(True)
plt.show()

Conclusion / Observations

  • Setosa flowers are clearly separable due to their small petal length.
  • Virginica flowers tend to have larger measurements overall.
  • Sepal length and petal length are positively correlated.
